# Study Notes Hub

**Author:** Elvis Cyusashema

## Description
This repository contains beginner study notes for version control practice. It was created as part of a classroom challenge to learn Git and GitHub workflows.

## Topics Covered
- Git basics (init, add, commit, push)
- GitHub workflow (clone, branch, pull request, merge)
- Collaboration best practices
- Version control concepts

## How to Use This Repository
1. Open the files and review the notes
2. Check the commit history to see how the project evolved
3. Explore the branches to understand feature development
